== Description ==

BeyondSEO helps businesses improve their online visibility directly from WordPress.

The plugin combines content optimization, technical SEO tools, local visibility features, review management, social media workflows, ads and connected marketing tools in one streamlined WordPress experience.

Unlike traditional SEO plugins focused only on metadata, BeyondSEO also helps businesses manage visibility across search engines, local listings, reviews, social platforms, and digital marketing channels.

The plugin includes built-in SEO functionality that works directly inside the WordPress editor. Users can optionally connect a free rankingCoach account to access additional visibility and marketing features.

== Data Privacy and Security ==

No data is transmitted to rankingCoach servers unless the user explicitly connects a rankingCoach account and grants permission.

BeyondSEO does NOT communicate with external servers by default. Backend communication is fully opt-in and only occurs after explicit user consent.

During onboarding, users can optionally allow the plugin to communicate with rankingCoach services available at:
https://www.rankingcoach.com

This setting can be changed at any time via:

Settings → General Settings → "Allow communication with rankingCoach services"

When enabled, the following information may be processed solely for support, compatibility, security verification, and connected feature functionality:

* Site information and domain
* WordPress and PHP versions
* Plugin configuration and account identifiers
* Request metadata (timestamp, request ID, user agent, IP address)

All data transmissions are secured using encrypted connections and HMAC-SHA256 request signing.

When deactivating the plugin, users can choose to delete all project data and settings associated with their connected rankingCoach project.

For full details:
https://www.rankingcoach.com/en-us/privacy-policy
